## 吉林大学软件工程软构件与中间件课设（核酸检测系统）
#### 利用JAVAEE的技术栈开发一个核酸检测系统。不关注前端页面美化，主要是写交互的逻辑。用到的主要技术为JSP，CSS，Servlet，JDBC。用到的JAVAEE特有的规范为有状态会话Bean，MDB，JMS，JPA。
